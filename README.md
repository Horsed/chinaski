chinaski (work in progress)
========

```<img src="localhost:9001/chinaski.gif?any=param" />```

Web app with a tracking pixel that stores any data in a MongoDB.

## Installation
  $ npm install chinaski

## Config
```config.js```

```js
  {
    mongodb: 'mongodb://127.0.0.1:27017/tracking',
    port: '9001'
  }
```
